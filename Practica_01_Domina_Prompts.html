Practica 1: Domina los Prompts

Aprende a comunicarte efectivamente con la IA

Duracion: 30-45 minutos | Nivel: Basico

Objetivo de la Practica

Al completar esta practica podras:
  • Identificar las 4 partes de un prompt efectivo
  • Transformar prompts vagos en prompts especificos
  • Usar Gemini para obtener respuestas utiles y accionables
Recuerda

La calidad del prompt determina la calidad de la respuesta. Un prompt vago = respuesta generica. Un prompt especifico = respuesta util.

Parte 1: La Formula del Prompt Perfecto

Todo buen prompt tiene 4 componentes. Memoriza esta formula:

ROL + CONTEXTO + TAREA + FORMATO (Quien) + (Situacion) + (Que quiero) + (Como lo quiero)

Ejemplo practico:

MAL
"Escribeme sobre marketing"

Problema: No tiene ROL, ni CONTEXTO, ni FORMATO. La IA no sabe que tipo de marketing, para quien, ni como presentarlo.

BIEN
"Actua como un consultor de marketing digital con 10 anios de experiencia. [ROL] Contexto: Tengo una cafeteria artesanal que quiere atraer mas clientes jovenes (18-30 anios). [CONTEXTO] Tarea: Dame 5 ideas de contenido para TikTok. [TAREA] Formato: Lista numerada, cada idea con titulo + descripcion de 2 lineas." [FORMATO]

Resultado: Respuesta especifica, util y lista para usar.

Ejercicio 1: Transforma el prompt malo

Abre gemini.google.com y prueba estos dos prompts:

Paso 1 - Prueba el prompt malo:

"Escribeme un email"

Observa la respuesta. Es generica y no te sirve directamente.

Paso 2 - Ahora prueba este prompt mejorado:

"Actua como un profesional de recursos humanos. Contexto: Necesito enviar un email a un candidato que entrevistamos ayer. El candidato se llama Carlos Mendez, aplico para el puesto de Analista de Datos. Nos gusto su perfil pero queremos hacerle una segunda entrevista. Tarea: Escribe un email profesional pero amigable invitandolo a una segunda entrevista el proximo martes a las 10 AM. Formato: Email corto (maximo 150 palabras), con saludo, cuerpo y despedida."

Ejercicio 2: Crea tu propio prompt

Piensa en algo que necesites para tu vida real (un email, una explicacion, una lista de tareas, etc.)

  1. Escribe primero un prompt "malo" (vago, sin detalles)
  2. Pruebalo en Gemini y observa la respuesta
  3. Ahora mejoralo usando la formula ROL + CONTEXTO + TAREA + FORMATO
  4. Prueba el prompt mejorado y compara
Ideas si no se te ocurre nada:
  • Email para pedir un aumento o permiso
  • Explicacion de un concepto que no entiendas
  • Plan de estudio para aprender algo nuevo
  • Lista de compras para una receta especifica

Ejercicio 3: Identifica las partes

Lee este prompt y marca que partes tiene:

"Eres un chef profesional especializado en cocina mediterranea. Tengo invitados este sabado y uno de ellos es vegetariano. Necesito un menu de 3 tiempos (entrada, plato fuerte, postre) que sea vegetariano pero que tambien le guste a los que comen carne. Dame las recetas en formato de lista con ingredientes y pasos numerados."

Entrega de la Practica

Para completar esta practica debes entregar:

  1. Screenshot del Ejercicio 1: Mostrando ambas respuestas de Gemini (prompt malo vs prompt bueno)
  2. Tu prompt personalizado (Ejercicio 2):
    • El prompt "malo" original
    • El prompt "bueno" mejorado
    • Screenshot de la respuesta de Gemini al prompt mejorado
  3. Respuesta escrita: Explica en 2-3 oraciones por que la formula ROL + CONTEXTO + TAREA + FORMATO funciona mejor que un prompt vago.
Criterios de evaluacion:
  • El prompt mejorado debe tener las 4 partes de la formula
  • La respuesta de Gemini debe ser notablemente mejor que con el prompt malo
  • La explicacion demuestra comprension del concepto
